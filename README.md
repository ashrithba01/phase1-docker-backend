# Phase 1 - Dockerized Node.js Backend

## 📌 Project Description
This project demonstrates containerizing a simple Node.js API using Docker.

## 🛠 Tech Stack
- Node.js
- Express
- Docker

## 🚀 How to Run Locally

1. Install dependencies:
npm install

2. Start application:
npm start

3. Open browser:
http://localhost:3000

## 🐳 Run Using Docker

1. Build Docker image:
docker build -t phase1-backend .

2. Run container:
docker run -p 3000:3000 phase1-backend

3. Open browser:
http://localhost:3000
